
SPM [ Snooker Player Profile Manager ]
=====================================

HowTo
----------------------------------------
- Compile code
```
git clone [https://github.com/tripples/spm.git](https://github.com/tripples/spm.git)
cd spm
git checkout $(tag)
mkdir build
cd build
cmake ..
make
cd ..; ./run arg.video
```
- Read doc more help.
