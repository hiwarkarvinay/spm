SPM ( Snooker Profile Management )
===========================

HowTo
--------------------
* Compile code

```
$ git clone https://github.com/tripples/spm.git ; cd spm                                        
$ git checkout your_tag
$ mkdir build ; cd build                                      
$ cmake ..                                      
$ make 
$./snooker video_Arg
```

* Read doc more help.
